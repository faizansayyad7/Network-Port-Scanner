# 🌐 Network Port Scanner

A web-based cybersecurity project built using Flask and Python that scans a target host to identify open ports and common running services. The application uses multithreading to improve scanning speed and presents results in a user-friendly cyber-themed dashboard.

---

## 🚀 Features

✅ Scan domains and IP addresses

✅ Detect open ports

✅ Identify common services

* FTP
* SSH
* HTTP
* HTTPS
* DNS
* SMTP
* MySQL
* Remote Desktop
* and more

✅ Multithreaded scanning

✅ Domain to IP resolution

✅ Interactive web interface

✅ Security scan summary

---
